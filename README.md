# 🍕 Pizza Order System (Desktop Application)

A lightweight and interactive desktop application for ordering pizza, developed using **C#** and **Windows Forms (.NET)**. This project was built as part of the structural programming and OOP roadmap by **Dr. Mohammed Abu-Hadhoud** (Programming Advices).

## 🚀 Features
* **Size Selection:** Choose between Small, Medium, and Large with dynamic pricing.
* **Crust Type:** Options for Thin or Thick crust.
* **Toppings Manager:** Add multiple toppings (Extra Cheese, Mushrooms, Tomatoes, Onions, Olives, Green Peppers) with instant price updates.
* **Where to Eat:** Select between Dine-In or Take-Away.
* **Order Summary:** Displays a complete, real-time receipt of the selected choices.
* **Reset Functionality:** Easily reset the form to place a new order.

## 🛠️ Tech Stack
* **Language:** C#
* **Framework:** .NET / Windows Forms (WinForms)
* **IDE:** Visual Studio
